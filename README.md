## Description - Connectly Frontend

Connectly is live chat web application inspired by WhatsApp. Users can create groups, send messages, and share files to their friends in real-tim

The Connectly app is built with [ViteJS](https://vitejs.dev/) in ReactJS, using TypeScript, [TailwindCSS](https://tailwindcss.com/) and [shadcnUI](https://ui.shadcn.com/).
It uses [Socket.IO](https://socket.io/) to have chat in real-time. 

It uses [Auth0](https://auth0.com/) for authentication and authorization. 

## Installation

You need to fill the env.sample for your own value

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run dev
```
